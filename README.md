# Acerte ou Forca

Este aplicativo foi desenvolvido como parte de uma atividade de avaliação da disciplina Algoritmos e Lógica de Programação do curso de Análise e Desenvolvimento de Sistemas da Fatec Zona Sul.

O principal objetivo foi criar um jogo da forca para praticar laços de repetição, arrays, estruturas de decisão e aplicar os conhecimentos adquiridos durante as aulas.

O jogo armazena as palavras em arquivos .txt que estão no projeto, e em seguida as lê e carrega em um array para realizar a lógica do jogo.

## Para executar o programa:
- Baixe o projeto.
- Vá até o diretório: `\acerte-ou-forca\bin\Debug\net8.0-windows`
- Execute o arquivo .exe localizado nessa pasta.

# Versão alternativa: Acerte ou Forca - Java

Uma versão alternativa, desenvolvida em Java, pensada para rodar em outros SOs além de Windows. Possui os mesmos recursos da versão escrita em C#, com as particularidades estéticas das ferramentas escolhidas.

### Para executar o programa:

- Abra o terminal e uso o comando `java --version` para descobrir a versão do Java instalada na maquina.
- Se não possuir JRE ou JDK instalado, necessario baixar via site da Oracle ou OpenJDK.
- Baixe o projeto.
- Vá até o diretório: `Acerte-ou-forca/AOF_Java_version/out/artifacts/AcerteOuForca_Java_version_jar`
- Execute o arquivo .jar localizado nessa pasta.
